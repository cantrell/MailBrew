MailBrew is a cross-platform, multi-protocol message notifier. It currently supports:

  • IMAP (encrypted and non-encrypted)
  • Gmail (including Google Apps email accounts)
  • Google Wave
  • Google Voice (both SMS and voicemail)

(Note that Google Wave and Google Voice support is experimental; there is no official API for either of these services yet.)
